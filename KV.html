Контрольные вопросы

1. Что такое «система контроля версий»?

Система контроля версий — это инструмент, который помогает отслеживать изменения в коде и управлять версиями файлов, позволяя нескольким разработчикам работать над проектом одновременно.

2. К какому типу VSC относится? В чем главное отличие от других типов?

Git относится к распределённым системам контроля версий. Главное отличие заключается в том, что каждая копия репозитория содержит полную историю изменений, а не только последнюю версию, как в централизованных системах.

3. В чем заключается преимущество использования Git?

Преимущества использования Git включают возможность работы в оффлайн-режиме, высокая скорость операций, возможность создания веток для разработки новых функций и легкость в слиянии изменений.

4. Какие команды Git вы знаете?

Некоторые основные команды Git: git init, git add, git commit, git push, git pull, git status, git branch.

5. В чем отличие Git и GitHub?

Git — это система контроля версий, а GitHub — это веб-сервис для хостинга Git-репозиториев, который предоставляет дополнительные функции, такие как управление проектами, совместная работа и отслеживание проблем.
